### Часть 2. Результаты решения ad hoc задач
#### 2.1. Существует ли зависимость активности игроков по совершению внутриигровых покупок от расы персонажа?

По доле платящих от общего количества совершивших покупки показатели по расам примерно одинаковые 16-20%. По средней сумме одной покупки наиболее высокие показатели у  Northman(761) и Elf (682), наиболее низкие показатели у расы  Human (403)
